# Divix — генеративная студия из шести воркспейсов

**Дата:** 2026-07-01
**Статус:** утверждён

## Цель

Перенести шесть ручных p5.js-инструментов (antlii.github.io) в единое приложение **Divix** по архитектуре Ritmo/Grafema: новый UI, оптимизация и фикс ошибок прошлых версий по ходу порта.

## Архитектура

Точная копия архитектуры Ritmo:

- **Vite SPA + PWA**, ES-модули, деплой на GitHub Pages через Actions.
- **p5.js 1.11.2 в instance mode** — каждый воркспейс — изолированный скетч `{name}Sketch`, монтируется через `new p5(sketch, container)`.
- **Ленивая загрузка** — воркспейсы регистрируются data-driven списком в `src/js/main.js`, код-сплит, чанк грузится на первую активацию вкладки. Тяжёлые вендорные библиотеки (paper, colorjs, h264-mp4-encoder) — per-workspace через `shared/utils/lazyLibs.js`.
- **Общая дизайн-система** — `src/css/style.css`, декларативные панели `shared/ui/panelBuilder.js` (SECTIONS-данные вместо Tweakpane 3 оригиналов).
- **localStorage-автосейв** per workspace, светлая/тёмная тема, PNG/MP4 экспорт через shared-утилиты, JSON импорт/экспорт пресетов.

Фундамент копируется из ritmo и адаптируется: `src/shared/`, `src/css/`, `src/js/main.js` (без auth/paywall), `vite.config.js`, eslint/prettier/vitest-конфиги, `public/lib`, workflow деплоя.

## Этапы

- **Этап 0 — каркас:** фундамент из ritmo, шесть пустых вкладок, README, первый коммит, рабочие `npm run dev/build/lint/test`.
- **Этапы 1–6 — по одному воркспейсу** в порядке вкладок: исходники инструмента скачиваются с antlii.github.io в локальную референс-папку (в git не попадает); порт: глобальные скрипты → ES-модули instance mode, Tweakpane UI → SECTIONS для panelBuilder, пресеты → JSON, экспорт → shared-утилиты. Каждый воркспейс проверяется визуально в dev-сервере против живой версии; замеченные баги оригинала фиксятся по ходу.
- **Вне рамок этого спека:** auth/paywall (Supabase + Stripe) — отдельный этап позже, по образцу Ritmo Pro.

## Обработка ошибок и тесты

- Чистые утилиты (map2, ease, геометрия, палитры) — покрытие vitest.
- `npm run lint` — ноль предупреждений (enforced).
- Сразу закладывается защита от классов багов, чинившихся в ritmo после ребилда: выход индекса палитры за границы при noise = 1, невалидные hex из генеративных палитр, гонки при асинхронной загрузке пресетов (draw-цикл гейтится до загрузки), фолбэки на невалидные массивы цветов.

## Деплой

Репозиторий `github.com/legenki/divix`, GitHub Pages через Actions → `legenki.github.io/divix`, PWA offline-ready.

## Критерии успеха

- Все шесть воркспейсов визуально воспроизводят поведение оригиналов (сравнение с antlii.github.io).
- Пресеты оригиналов перенесены и работают.
- PNG/MP4 экспорт работает во всех воркспейсах.
- Сборка, линт и тесты проходят чисто; деплой на Pages живой.
